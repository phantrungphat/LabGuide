Câu 1: Có những cách đặt Javascript nào trong file HTML? Cho ví dụ minh họa?

Tương tự như thêm CSS, để thêm Javascript vào trong file HTML, chúng ta có 3 cách:

Cách 1: Internal - viết trong file HTML hiện tại


Viết đoạn mã Javascript trong cặp thẻ <script></script>. Đoạn mã thường được đặt tại phần head, tuy nhiên đây không phải là điều bắt buộc, bạn có thể đặt ở đâu tùy thích, miễn là nằm trong cặp thẻ <script></script>.

				   	<!--Đặt tại thẻ head -->
						<html>
						    <head>
						        <title></title>
						        <script language="javascript">
						            alert("Hello World!");
						        </script>
						    </head>
						    <body>
						         
						    </body>
						</html>
						
				   	<!--Đặt tại thẻ body -->
						<html>
						    <head>
						        <title></title>
						    </head>
						    <body>
						         <script language="javascript">
						            alert("Hello World!");
						        </script>
						    </body>
						</html>

Cách 2: Inline - viết trực tiếp trong thẻ HTML

Cách này có nghĩa là viết trực tiếp Javascript vào trong thẻ HTML. 
			
			<!-- Khi nhấn vào button thì hiển thị thời gian hiện tại-->
			<button type="button"
                		onclick="document.getElementById('hien-thi-thoi-gian-ra').innerHTML = Date()">Button</button>
            		<p id="hien-thi-thoi-gian-ra"></p>

Cách 3: External - viết ra một file js khác rồi import vào

Đối với cách này, bạn phải tạo một file có phần mở rộng là .js, sau đó dùng thẻ script để import vào. Lúc này, bên trong file .js không cần phải có thẻ script nữa vì đã có phần mở rộng là .js nên trình duyệt sẽ tự động nhận diện đây là file chứa mã Javascript. 
			<script language="javascript" src="Ahihi.js"></script>

Câu 2: Có thể hiển thị dữ liệu từ Javascript với những cách nào? Cho ví dụ minh họa.

Javascript có thể hiển thị dữ liệu theo nhiều cách khác nhau.

  • Viết vào bên trong thẻ HTML, sử dụng innerHTML
  • Viết lên trên trình duyệt, sử dụng document.write()
  • Viết theo hộp thoại thông báo, sử dụng window.alert()
  • viết lên browser console, sử dụng console.log()

Đầu tiên, sử dụng innerHTML

Để truy cập đến một phần tử HTML, chúng ta phải sử dụng phương thức document.getElementById(id).
id ở đây dùng để xác định thẻ HTML nào, innerHTML dùng để định nghĩa nội dung của thẻ HTML.

Ngoài ra, còn có các lựa chọn khác như innerText,innerContent.

	
				
			<body>
				<p>Chèn chữ Ahihi vào thẻ p có id là display</p>

				<p id="display"></p>

				<script>
					document.getElementById("display").innerHTML = "Ahihi"; 
				</script>
			</body>

Tiếp theo là document.write()

Khi sử dụng document.write(), nội dung mà bạn muốn hiển thị sẽ được hiển thị ngay đúng vị trí mà nó được đặt trong trang web.

			
				<!DOCTYPE html>
				<html>
					<body>
				<!-- Đối với đoạn code này thì chương trình sẽ in ra tuần tự đúng như những gì ta mong muốn -->
    						<h1>Dòng đầu tiên nè!</h1>

    						<script>document.write("<h1>Ahihi 1</h1>");</script>

    						<h1>Để cho nó cách 2 cái ra nè!</h1>

    						<script>document.write("<h1>Tài liệu học MySQL</h1>");</script>

   		 				<script>document.write("<h1>Tài liệu học PHP</h1>");</script>

					</body>
				</html>

document.write() thường được sử dụng trong khâu testing, nếu nó được sử dụng khi trang web đã tải hoàn toàn thì nó sẽ xóa tất cả các phần tử HTML.

				
<!-- Đối với đoạn code này sau khi chạy thì màn hình chỉ có dòng chữ Ahihi khi ta bấm button bởi vì trang web đã tải hoàn tất và chúng ta sử dụng document.write() để đè lên.>
			<!DOCTYPE html>
			<html>
				<body>
					<h1>Dòng này sẽ không hiển thị 1</h1>
				 	<p>Dòng này sẽ không hiển thị 2</p>
					<button type="button" onclick="document.write("Ahihi")">Button</button>
				</body>
			</html>

Tiếp đến là sử dụng hàm alert()

Hàm alert() có nhiệm vụ in một thông báo popup, có tham số truyền vào là nội dung ta muốn thông báo với người sử dụng.

Ngoài ra còn có confirm() và prompt()

. 
		<input type="button" onclick="alert('Xin chào các bạn')" value="Click Me"/>

Cuối cùng là console.log()

Hàm console.log() thường được sử dụng để debug, nó có nhiệm vụ show ra giá trị của tất cả các loại dữ liệu như number, integer, array, object,... 

		<!-- Chúng ta cần truy cập đến tab Console trong Website Debug để xem giá trị của biến website được in ra -->
				var website = 'Ahihi.com';
					console.log(website);

Câu 3: Làm thế nào để truy xuất (lấy giá trị) 1 thẻ HTML bất kỳ theo ID, theo Class, theo Name, theo tên thẻ HTML và theo CSS Selector? Cho ví dụ minh họa.

Chúng ta sẽ sử dụng các phương thức sau để truy xuất đến các đối tượng:

  • document.getElementById()
  • document.getElementsByClassName()
  • document.getElementsByName()
  • document.getElementsByTagName()

Sau đó sử dụng innerHTML để lấy nội dung hoặc sử dụng document.CacPhuongThucOTren().attributeName để lấy giá trị của đối tượng đã truy xuất.

document.getElementById()

Phương thức document.getElementById() là một trong những phương thức phổ biến nhất của HTML DOM, nó được sử dụng rất thường xuyên để thao tác hoặc lấy thông tin của form hoặc một phần tử bất kỳ.

Phương thức sẽ trả về Null nếu không có phần tử nào có thuộc tính ID được cung cấp.

Phương thức getElementById() sẽ trả về phần tử có thuộc tính ID được cung cấp.

							
		<html>
    			<body>
        			<script language="javascript">
		          		function change()
		          		{
		             			// Lấy đối tượng
		             			var object = document.getElementById("object");
		         
		             			// lấy thuộc tính type
		             			var type = object.type;
				
		             			// kiểm tra thuộc tính type và thay đổi
		            			
		            			if (type == "button"){
		          
		               				object.type = 'text';
		            			}
		            			else{
		                			object.type = "button";
		            			}
		        
		          		}
		        	</script>
		        	<input type="button" value="CLick me" onclick="change()" id="object" />
		    	</body>
		</html>
document.getElementsByClassName()

Tương tự document.getElementById(), phương thức document.getElementsByClassName() sẽ trả về tập hợp các phần tử trong trang có thuộc tính class được cung cấp khi gọi phương thức. Kết quả sẽ được trả về dưới dạng một đối tượng NodeList object.

							
			<body>
		            <h6>Ahihi</h6>
		            <h6 class="demo"> Số 1</h6>
		            <h6 class="demo"> Số 2</h6>
		            <h6 class="demo"> Số 3</h6>
		        	<button onclick="changebackgroundcolor()">Xem kết quả</button>
		        	<script>
		            	function changebackgroundcolor(){
		                	
		                	var NodeObject = document.getElementsByClassName("demo");
		                	NodeObject[0].style.color = "red";
		                	NodeObject[1].style.color = "blue";
		                	NodeObject[2].style.color = "yellow";
		                	
		            	}
		        	</script>
    			</body>

document.getElementsByName()

Phương thức này có phần tương tự phương thức document.getElementsByClassName(), sẽ trả về tập hợp các phần tử trong trang có thuộc tính name được cung cấp khi gọi phương thức. Kết quả sẽ được trả về dưới dạng một đối tượng NodeList object.

Cú pháp: document.getElementsByName(name)

name ví dụ như: thẻ h3 có name=demo,...

document.getElementsByTagName()

Phương thức này sẽ trả về tập hợp các phần tử trong trang có tên thẻ được cung cấp khi gọi phương thức. Kết quả sẽ được trả về dưới dạng một đối tượng NodeList object.

Bạn có thể truyền tham số * để lấy tất cả các phần tử trên trang.

Cú pháp: document.(tagname)

tagname ví dụ như: h3, h1, li,...

				        function get() {
				        <!-- Sử dụng Date Object trong Javascript -->
				            var date = new Date();
				            var data += 'Năm đầy đủ: ' + date.getFullYear() + '<br />';
				            data += 'Tháng: ' + date.getMonth() + '<br />';
				            data += 'Ngày: ' + date.getDate() + '<br />';
				            data += 'Giờ: ' + date.getHours() + '<br />';
				            data += 'Phút: ' + date.getMinutes() + '<br />';
				            data += 'Giây: ' + date.getSeconds() + '<br />';
				            document.getElementById('result').innerHTML = data;
				        }
				    </script>

						<!-- Ở đây ta sử dụng phương thức Math.random() để random con số từ 0 đến 1 -->

						<!-- Nhân 1000 và cộng 1 giúp ta điều chỉnh khoảng random chính xác -->
